Technologies for web development

One of the first things that I want Academis to have is a comfortable and informative website where I can put tips and tricks for scientists and of course write about tutorials I offer. After a few rounds of consulting with Justyna, we came up with five different tools that we want to use for developing the website.

  • Web server: We decided the website would be written in the Django framework. This is a toolkit for creating websites in the Python programming language. We picked it because we were both familiar with it.
  • Special effects: We also decided to use jQuery, a JavaScript library for making sites easier to navigate.
  • Hosting: Next we needed a place to host the website. We found DjangoEurope to provide everything required at attractive conditions (cost of course, but also support by Django experts).
  • Version control: To share the source code among developers and the server, we created a repository on Bitbucket that can be used with Mercurial.
  • Project management: And finally, to keep track of tasks in the project we use Kanbanery.

Of course, to each of these points there are many alternatives. But it was important to make a choice in order to move on.
